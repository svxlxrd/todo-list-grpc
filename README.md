# Todo List gRPC MVP

Это минималистичный пример реализации микросервиса **Todo List** с использованием **gRPC** и **Protocol Buffers** на языке Go. Проект демонстрирует базовый CRUD-функционал (Create, Read, Update, Delete) в рамках клиент-серверной архитектуры.

## 🚀 Основные возможности
* **gRPC API**: Быстрое взаимодействие через бинарный протокол (HTTP/2).
* **CRUD операции**: Полный цикл работы с задачами.
* **Thread-safe**: Использование `sync.RWMutex` для безопасной работы с данными в памяти.
* **Контракт-ориентированность**: Использование `.proto` файлов для описания структуры данных.

## 🛠 Технологический стек
* **Language**: Go 1.22+
* **Protocol Buffers**: v3
* **gRPC**: `google.golang.org/grpc`
